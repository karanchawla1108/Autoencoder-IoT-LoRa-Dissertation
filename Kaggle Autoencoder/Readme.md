# Kaggle Autoencoder — How to Run
link: https://www.kaggle.com/datasets/puneet6060/intel-image-classification
 
This folder contains the autoencoder trained on the **Intel Image Classification** dataset from Kaggle.
 
---
 
## Before You Start — Get Your Kaggle API Key
 
1. Go to [kaggle.com](https://www.kaggle.com) and log in
2. Click your **profile picture** (top right) → **Settings**
3. Scroll down to the **API** section
4. Click **Create New Token**
5. A file called `kaggle.json` will download — keep it safe
 
It will look like this:
```json
{
  "username": "your_kaggle_username",
  "key": "your_api_key_here"
}
```
